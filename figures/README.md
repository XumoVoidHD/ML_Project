# RUL Figures

## Preprocessing (EDA)

Generated by `preprocess_rul.py` or `preprocess_visualize.py`:

| File | Description |
|------|-------------|
| `preprocess_capacity_degradation.png` | Capacity vs cycle per battery (degradation curves) |
| `preprocess_rul_distribution.png` | RUL histogram (train vs test) |
| `preprocess_correlation_heatmap.png` | Feature correlation matrix |
| `preprocess_feature_distributions.png` | Histograms of key features |
| `preprocess_impedance_trends.png` | Re and Rct over cycles per battery |

## Model Evaluation

Each training script produces 5 figures:

| File pattern | Description |
|--------------|-------------|
| `*_actual_vs_predicted.png` | Scatter: actual vs predicted RUL (train + test) |
| `*_residuals.png` | Residuals vs predicted + residual histogram |
| `*_feature_importance.png` | Bar chart of feature importance |
| `*_rul_over_cycles.png` | Actual vs predicted RUL over cycle index (test battery) |
| `*_confusion_matrix.png` | Binned RUL: Early/Mid/Late (predicted vs actual) |

**Prefixes:** `baseline`, `xgboost`, `gradient_boosting`, `ensemble`, `tuned`, `extended_features`
